## 🇸🇦 نبذة

أداة سريعة وذكية لإنشاء روابط واتساب مباشرة للأرقام السعودية — بتصميم **Neon Pulse** العصري، تعمل بدون إنترنت، وتُثبَّت كتطبيق على جهازك.

<br>

## ✨ المميزات

| | الميزة | الوصف |
| :---: | :--- | :--- |
| 🧠 | **تعرّف ذكي على الأرقام** | يقبل أي صيغة: `05XX` أو `+966 XX` أو `00966...` — بمسافات أو شرطات أو نقاط |
| 📲 | **تثبيت كتطبيق** | يعمل كتطبيق مستقل على جوالك أو كمبيوترك بدون متصفح |
| 📡 | **يعمل بدون إنترنت** | جميع الوظائف تعمل offline بعد أول زيارة |
| 💾 | **حفظ تلقائي** | يحفظ آخر 5 روابط مع إمكانية مسح السجل بالكامل |
| 📤 | **مشاركة فورية** | شارك الرابط مباشرة أو انسخه بضغطة واحدة |
| 🔍 | **استعلام عن الرقم** | زر مباشر للاستعلام عن أي رقم سعودي |
| ✨ | **تأثيرات بصرية** | جسيمات متحركة، كونفيتي، وتصميم زجاجي (Glassmorphism) |
| 📱 | **متجاوب بالكامل** | يعمل على كل الشاشات من الجوال حتى 4K |
| ♿ | **وصولية كاملة** | دعم قارئات الشاشة، التنقل بالكيبورد، وتباين ألوان WCAG AA |

## 🛠️ طريقة الاستخدام

| | الخطوة |
| :---: | :--- |
| 1️⃣ | **أدخل الرقم** — اكتب أو الصق رقم الجوال السعودي بأي صيغة |
| 2️⃣ | **أنشئ الرابط** — اضغط زر **إنشاء الرابط** 🔗 |
| 3️⃣ | **استخدم الرابط** — انسخه أو شاركه أو افتح المحادثة مباشرة |

> 💡 يدعم الأرقام العربية `٠٥٥٩٩٤٨١٤٩` والفارسية `۰۵۵۹۹۴۸۱۴۹` تلقائياً

<br>

## 📱 صيغ الأرقام المدعومة

الأداة تتعرّف تلقائياً على جميع طرق كتابة الرقم السعودي:

```
 0512345678             ✅  محلي
 05 123 45678           ✅  محلي بمسافات
 051-234-5678           ✅  بشرطات
 +966 51 234 5678       ✅  منسوخ من واتساب
 +966512345678          ✅  دولي
 00966 51 234 5678      ✅  دولي بـ 00
 (05) 123-45678         ✅  بأقواس
 05.123.45678           ✅  بنقاط
```

<br>

## 📲 التثبيت كتطبيق

| Android / Chrome | iPhone / Safari |
| :--- | :--- |
| اضغط على زر **«تثبيت كتطبيق»** الذي يظهر تلقائياً أسفل الصفحة | اضغط **مشاركة** ↗ ثم **«إضافة إلى الشاشة الرئيسية»** |

بعد التثبيت، التطبيق يظهر كأيقونة على شاشتك الرئيسية ويعمل بدون متصفح وبدون إنترنت.
